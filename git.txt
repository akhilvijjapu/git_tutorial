git config --global user.name : This command will give us the user name
git config --global user.email : This command will give us the user email
git status : This command will give us the committed and uncommited file status(untracked files).
git add .: This command will add all the files to staging directory waits for the commit.
git commit -m "commit description": This command will commit all the files to staging directory
git branch : This command will show the current branch
git remote -v : This command will show the the current git repository mapping to.
git remote add origin "git_url": This command will add the git repository to be mapped.
git push origin main: This command will push all the files from staging directory to git repository main branch.
git clone "url": This command will clone the entire repository.
git restore: This command will restore the changes if any anything made.
